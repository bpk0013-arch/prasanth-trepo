## üé® Screenshot Best Practices

### Quality Standards

1. **Resolution**
   - Minimum: 1920x1080 for full screens
   - Minimum: 800x600 for specific sections
   - Format: PNG (lossless)
   - DPI: 72 for web, 300 for print

2. **Clarity**
   - Text must be readable
   - No blur or pixelation
   - Proper lighting (not too dark/bright)
   - High contrast

3. **Composition**
   - Center the subject
   - Remove distractions
   - Use whitespace effectively
   - Align elements properly

4. **Consistency**
   - Same theme across all screenshots
   - Same sample data where possible
   - Same window size/zoom level
   - Same color scheme

### Tools Recommended

**For Screenshots:**
- Windows Snipping Tool (built-in)
- ShareX (free, powerful)
- Greenshot (free, annotations)
- Lightshot (free, easy)

**For Editing:**
- GIMP (free, powerful)
- Paint.NET (free, simple)
- Photoshop (paid, professional)
- Canva (free/paid, easy)

**For Annotations:**
- Snagit (paid, best)
- Skitch (free, simple)
- Annotate (free, basic)
- Greenshot (free, built-in)

**For GIFs:**
- ScreenToGif (free, excellent)
- LICEcap (free, simple)
- Gifox (Mac, paid)
- Peek (Linux, free)

**For Banners:**
- Canva (easiest)
- Figma (professional)
- Adobe Spark (good)
- Crello (alternative)

### 2. GUI Main Interface

**Steps:**
1. Open Prasanth_TRepo_GUI_v2.exe
2. Fill in sample data:
   - Input: "sample_results.jtl"
   - Project: "E-commerce Load Test"
   - Environment: "Production"

### 3. PDF Screenshots

**Steps:**
1. Generate a report with good sample data
2. Open PDF in Adobe Reader or browser
3. Zoom to 100%
4. For each section:
   - Navigate to section
   - Take screenshot
   - Crop to relevant area
   - Save with descriptive name

**Sections to capture:**
- Cover page (full page)
- Executive summary (full page)
- Charts section (all 3 charts visible)
- Detailed results (table with data)
- Error analysis (with error codes)

### 4. Excel Screenshots

**Steps:**
1. Open generated Excel file
2. Maximize Excel window
3. For each sheet:
   - Select the sheet
   - Zoom to 100%
   - Take screenshot
   - Save with descriptive name

**Sheets to capture:**
- Summary (with logo visible)
- Detailed Results (with data)
- Charts & Analysis (with charts)

**Steps:**
1. Create split-screen image
2. Left side: Raw JTL file in Notepad

4. Add labels: "Before" and "After"
5. Add arrow between them
6. Save as before_after.png
